# Myriad Online — Demo Client

A systems-driven 2D MMORPG. Download the latest client from [Releases](https://github.com/Bokhard/myriad-online-demo-client/releases).

## Quick Start

1. Go to [Releases](https://github.com/Bokhard/myriad-online-demo-client/releases)
2. Download the latest `.zip`
3. Extract and run **MyriadOnline.exe**
4. Select **Cloud** from the server dropdown
5. Create an account or log in
6. Pick a character and enter the world

## Requirements

- Windows 10/11
- GPU with OpenGL 3.3+ support

## About

Myriad Online is a 2D MMORPG with three interconnected gameplay spheres: Combat, Industrial, and Statecraft. This repo contains only the demo client binary — the game source is in a separate private repository.
