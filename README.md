# Myriad Online — Demo Client

A systems-driven 2D MMORPG with three interconnected gameplay spheres: Combat, Industrial, and Statecraft. Top-down perspective with field-of-view lighting, tab-target combat, and a procedurally generated world.

## Download & Play

1. Go to [**Releases**](https://github.com/Bokhard/myriad-online-demo-client/releases)
2. Download the latest `MyriadOnline-Client-*.zip`
3. Extract the zip
4. Run **MyriadOnline.exe**

## How to Connect

1. Launch the client
2. Select **Cloud** from the server dropdown
3. Create an account (email + password) or log in
4. Create a character or select an existing one
5. Enter the world

## Requirements

- Windows 10/11
- GPU with OpenGL 3.3+ support

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Aim / Look direction |
| F | Toggle auto-attack |
| Tab | Cycle targets |
| 1-8 | Action bar abilities |
| I | Inventory |
| C | Character sheet |
| M | Map |
| P | Ability book |
| L | Combat log |
| Enter | Chat |
| Esc | Menu |

## About

This repository hosts only the compiled demo client. The game source code is maintained in a separate private repository. The server runs on [Fly.io](https://fly.io) — no self-hosting needed to play.
